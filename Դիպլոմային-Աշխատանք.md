# ԳԼՈՒԽ 1․ ՎՈՐՈՆՈՎԻ ԴԻԱԳՐԱՄ

## 1.1 Նախնական հասկացություններ {#նախնական-հասկացություններ}

Ենթադրենք, սուպերմարկետների ցանցի զարգացման խորհրդատվական հանձնաժողովի կողմից կան պլաններ նոր խանութ բացել որոշակի վայրում: Կանխատեսելու համար, թե արդյոք նոր խանութը շահութաբեր կլինի, պետք է գնահատել, թե քանի հաճախորդ կգրավի այն: Դա անելու համար պետք է կառուցել պոտենցիալ գնորդների վարքագծային մոդել. ինչպե՞ս է մարդը որոշում, թե որտեղ գնալ գնումների: Նմանատիպ հարց է ծագում սոցիալական աշխարհագրության մեջ, երբ ուսումնասիրվում է երկրի տնտեսական գործունեությունը. ինչպիսինն է քաղաքների առևտրային տարածքը: Ավելի վերացական, մենք ունենք բազմաթիվ սպասարկման կենտրոններ, որոնք առաջարկում են ապրանքներ կամ ծառայություններ, և յուրաքանչյուր սպասարկման կենտրոնի համար մենք ցանկանում ենք իմանալ, թե որտեղ են ապրում այն մարդիկ, ովքեր ապրանքներ կամ ծառայություններ են գնում այդ կենտրոնից: Խնդիրն ուսումնասիրելու համար ներմուծենք հետևյալ պարզեցնող ենթադրությունները.

![](media/image1.jpeg){width="3.697674978127734in" height="2.9721270778652666in"}

Նկար 1.1 - Հոլանդիայի տասներկու նահանգների մայրաքաղաքների առևտրային գոտիները, որոնք կանխատեսվել են Վորոնովի մոդելի միջոցով։

- բոլոր կենտրոններն առաջարկում են նույն ապրանքները կամ ծառայությունները

> նույն գնով;

- ապրանքի կամ ծառայության ձեռքբերման արժեքը հավասար է դրա գնին

գումարած դեպի կենտրոն ուղևորության արժեքը;

- մինչև կենտրոն ուղևորության արժեքը հավասար է դրանից էվկլիդեսյան

- հեռավորությանը՝ բազմապատկած միավոր հեռավորության վրա ծախսված ֆիքսված արժեքի գնով;

- գնորդները ձգտում են նվազագույնի հասցնել ապրանքների կամ ծառայությունների գնման ծախսերը:

Սովորաբար, այս ենթադրությունները լիովին բավարարված չեն լինում. որոշ կենտրոններում ապրանքները կարող են ավելի էժան լինել, քան մյուսներում, և մի կետից մյուսը ճանապարհորդության արժեքը պարտադիր չէ, որ համեմատական լինի նրանց միջև էվկլիդեսյան հեռավորությանը: Բայց նկարագրված մոդելը մոտավոր տալիս է կոպիտ մոտարկում դեպի սպասարկման կենտրոնների առևտրային գոտիները։ Այն տարածքները, որտեղ բնակիչների վարքագիծը տարբերվում է մոդելի կողմից կանխատեսածից, կարող է ենթարկվել ավելի սերտ հետազոտության՝ հասկանալու համար, թե որն է շեղումների առաջացման պատճառը: \[1\]

Մեզ հետաքրքրում է այս մոդելի երկրաչափական մեկնաբանությունը: Ձևակերպված ենթադրությունները դրդում են ամբողջ դիտարկվող տիրույթի բաժանումը տարածաշըրջանների՝ սպասարկման կենտրոնների առևտրային գոտիների, այնպես, որ մի շրջանի բնակիչները գնումներ են կատարում նույն կենտրոնում: Մեր ենթադրություններից բխում է, որ մարդն ուղղակի ապրանք է գնում մոտակա կենտրոնում՝ իրավիճակը բավականին մոտ է իրականին։ Սա նշանակում է, որ տվյալ կենտրոնի առևտրային տարածքը բաղկացած է բոլոր կետերից, որոնց համար այս կենտրոնը ցանկացած այլ կենտրոնից ավելի մոտ է: Վերևում պատկերված նկարում սպասարկման կենտրոնները Նիդերլանդների տասներկու նահանգների մայրաքաղաքներն են:

Մոդելը, որում յուրաքանչյուր կետ պատկանում է մոտակա սպասարկման կենտրոնին, կոչվում է Վորոնովի մոդել, իսկ այս մոդելի կողմից առաջացած բաժանումը կոչվում է Վորոնովի դիագրամ կենտրոնների հավաքածուի համար: Վորոնովի դիագրամից մենք կարող ենք տարբեր տեղեկություններ քաղել կենտրոնների առևտրային տարածքների և դրանց փոխհարաբերությունների մասին: Օրինակ, եթե երկու կենտրոնների շրջաններն ունեն ընդհանուր սահման, ապա, ամենայն հավանականությամբ, այդ կենտրոնները մրցում են սահմանամերձ գնորդների համար։

Վորոնովի դիագրամը ամենուր է: Մենք նկարագրեցինք դրա կիրառությունը սոցիալական աշխարհագրության մեջ, սակայն կան կիրառություններ ֆիզիկայի, աստղագիտության, ռոբոտաշինության և շատ այլ առարկաների մեջ: Վորոնովի դիագրամները սերտորեն կապված են մեկ այլ կարևոր երկրաչափական կառուցվածքի՝ Դելոնի եռանկյանացման հետ: Այս գլխում մենք կսահմանափակվենք հարթության վրա մի շարք կետերի Վորոնովի դիագրամի հիմնական հատկություններով և դրանց կառուցումներով:

Ալգորիթմների բարդության, ինչպես նաև խնդիրների լուծման բարդության ստորին գնահատականների համար, օգտագործվում է O -- սիմվոլիկան (\[2\]): Դիցուք$\ g$--ն և $f$--ը ոչ բացասական ամբողջ թվերի բազմության վրա որոշված ոչ բացասական ֆունկցիաներ են: Այդ դեպքում

$$g(n) = O\left( f(n) \right)$$

գրառումը նշանակում է, որ գոյություն ունեն C և N~o~ \> 0 հաստատուններ այնպես, որ $\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ g(n) \leq c \bullet f(n)$

տեղի ունի բոլոր $n \geq N_{0}$համար: O սիմվոլը\` օգտագործվում է ֆունկցիայի աճի արագության վերին գնահատականների համար: Օրինակ\` $an + b = Օ(n)$ ( կարելի է վերցնել $C = a + 1$ և $N_{0} = b$), $g(n) = \Omega\left( f(n) \right)\ $գրառումը նշանակում է, որ գոյություն ունեն $C$ և $N_{0} > 0$ հաստատուներ այնպես, որ

$$g(n) \geq C \bullet f(n)$$

տեղի ունի բոլոր $n \geq N_{0}$ համար: $\Omega$ սիմվոլը օգտագործվում է ֆունկցիայի աճի արագության ստորին գնահատականների համար:

$$g(n) = \theta\left( f(n) \right)$$

գրառումը նշանակում է, որ գոյություն ունեն $C_{1}$,$C_{2}$և $N_{0} > 0$ հաստատուներ այնպես, որ

$$C_{1}f(n) \leq \left| g(n) \right| \leq C_{2}f(n)$$

տեղի ունի բոլոր n ≥ N~0~ համար: Կարելի է ցույց տալ որ g(n) = θ(f(n)) այն և միայն այն դեպքում, երբ g(n) = O(f(n)) և g(n) = Ω(f(n)):

## 1.2. Սահմանումը և հիմնական հատկությունները {#սահմանումը-և-հիմնական-հատկությունները}

Երկու p և q կետերի միջև էվկլիդյան հեռավորությունը նշանակենք dist(p,q):

Հարթության վրա գործում է հետևյալ բանաձևը.

$$dist(p,q) ≔ \sqrt{\left( p_{x}\  - \ q_{x} \right)^{2}\  + \ \left( p_{y}\  - \ q_{y} \right)^{2}}$$

Դիցուք P ≔ {p~1~, p~2~, ... , p~n~} բազմությունը n կետերի բազմություն է հարթության վրա; այդ կետերը կլինեն սպասարկման կենտրոնները։ Սահմանենք P-ի բազմության Վորոնովի դիագրամը՝ որպես հարթության բաժանում n բջիջների՝ մեկական p~i~ - ի յուրաքանչյուր կենտրոնի համար, օժտված այն հատկությամբ, որ q կետը պատկանում է p~i~ կենտրոնին համապատասխանող բջիջին, այն և միայն այն դեպքում, երբ $dist\left( q,\ p_{i} \right) < dist\left( q,\ p_{j} \right)$ ցանկացած $\ p_{i} \in P,\ j \neq i$ կետի համար. P բազմության Վորոնովի դիագրամը կնշանակենք Vor(P)-ով: Արտահայտությունների մեջ թույլատրելով որոշակի ազատություն, երբեմն կօգտագործենք «Vor(P) կամ Վորոնովի գծապատկեր» տերմինը՝ նշելով միայն տրոհման կողերն ու գագաթները: Օրինակ, երբ ասում ենք, որ Վորոնովի դիագրամը կապակցված է, նկատի ունենք, որ կողերի և գագաթների միավորումը կազմում է կապակցված բազմություն։ $p_{i}$ կենտրոնին համապատասխանող Vor(P) բջիջը նշանակվում է V($p_{i}$); մենք այն կանվանենք Վորոնովի բջիջ $p_{i}$ -ի համար (գլխի սկզբում ներկայացված տերմինաբանության մեջ V($p_{i}$) - ն $p_{i}$ -ի կենտրոնի առևտրային գոտին է):

Այժմ ավելի ուշադիր նայենք Վորոնովի դիագրամին: նախ և առաջ ուսումնասիրենք Վորոնովի մեկ բջիջի կառուցվածքը: Հարթության երկու p և q կետերի համար p - ի և q -- ի միջնուղղահայաց է կոչվում $\overline{pq}$ հատվածի միջնակետով անցնող և դրան ուղղահայաց ուղիղը: Միջնուղղահայացը հարթությունը բաժանում է երկու կիսահարթությունների։

Նշանակենք h(p, q) այն բաց կիսհարթությունը,որը պարունակում է p - ն, իսկ h(q, p) - ով բաց կիսհարթությունը, որը պարունակում է q - ն։ Նկատենք, որ r∈ h(p, q) այն և միայն այն դեպքում երբ $dist(r,p) < dist(r,q)$: Սրանից հետևում է հետևյալ դիտողությունը.

**Դիտողություն 1.1***.* $V\left( p_{i} \right) = \cap_{1 \leq j \leq n,\ j \neq i}h\left( p_{i},\ p_{j} \right)$

Այսպիսով, V(p~i~) -ն n - 1 կիսահարթությունների հատումն է, և, հետևաբար, ուռուցիկ բազմանկյուն տիրույթ է (հնարավոր է անսահմանափակ) առավելագույնը n - 1 գագաթներով և առավելագույնը n - 1 կողերով։

## 1.3. Վորոնովի դիագրամի հաշվարկ {#վորոնովի-դիագրամի-հաշվարկ}

*Նկար 1.13 - Կոտրվածքների գծից աղեղի անհետացումը.*

- Իրադարձությունների հերթականությունը 𝑄-ն իրականացվում է առաջնահերթ հերթականության միջոցով, որտեղ իրադարձության առաջնահերթությունը նրա 𝑦կոորդինատն է: Հերթականությունը պահպանում է առաջիկա իրադարձությունները, որոնք արդեն հայտնի են: Կենտրոնի իրադարձության դեպքում մենք պարզապես պահում ենք հենց կենտրոնը: Իսկ շրջանագծի իրադարձության համար մենք պահում ենք շրջանագծի ամենացածր կետը և ցուցիչը դեպի 𝑇 -- ի տերևային հանգույց, որը ներկայացնում է այն աղեղը, որը կանհետանա իրադարձության կատարման պահին:

Մաքրման գործընթացքում կոտրվածքների գծի տոպոլոգիական կառուցվածքը փոխվում է յուրաքանչյուր իրադարձության հետ: Երբեմն հայտնվում են հարևան աղեղների նոր եռյակներ, երբեմն էլ անհետանում են գոյություն ունեցողները։ Մեր ալգորիթմը պետք է ապահովի, որ կոտրվածքների գծի ցանկացած երեք հարակից աղեղների համար, որոնք սահմանում են շրջանագծի պոտենցիալ՝ իրադարձություն, այդ իրադարձությունը կտեղադրվի 𝑄 հերթականության մեջ: Այստեղ կան երկու նրբություններ: Նախ, կարող են լինել հարևան եռյակներ, որոնց համար երկու կոտրման կետեր չեն համընկնում, այսինքն՝ շարժվում են այնպիսի ուղղություններով, որոնք բացառում են ապագայում հանդիպումը: Դա տեղի է ունենում, երբ կոտրման կետերը շարժվում են հատման կետից հեռու տանող միջնուղղահայացի երկայնքով: Այս դեպքում եռյակը չի սահմանում շրջանագծի պոտենցիալ՝ իրադարձություն: Երկրորդ, նույնիսկ եթե եռակի կոտրման կետերը համընկնեն, համապատասխան շրջանագծի իրադարձությունը կարող է տեղի չունենալ. դա տեղի է ունենում, երբ եռյակը անհետանում է (օրինակ, կոտրվածքների գծի վրա նոր կենտրոնի առաջացման դեպքում) նախքան իրադարձությունը տեղի ունենալը: Նման իրադարձությունը մենք կանվանենք կեղծ ահազանգ։

**Լեմմա1.4.** *Նկարագրված ալգորիթմի իրականացման ժամանակը 𝑂(𝑛 𝑙𝑜𝑔 𝑛) է, իսկ սպառված հիշողության չափը՝ 𝑂(𝑛):*

**Ապացույց.** 𝑇 ծառի և իրադարձությունների 𝑄 հերթի վրա պարզունակ գործողությունները, մասնավորապես տարրի տեղադրումն ու հեռացումը, պահանջում են $O(\log n)$ ժամանակ: Պարզունակ գործողությունները կողերի կրկնակի կապակցված ցանկի վրա պահանջվում է հաստատուն ժամանակ: Իրադարձությունը կարգավորելու համար մենք կատարում ենք նման պարզունակ գործողություններ, ուստի մենք դրա վրա ծախսում ենք $O(\log n)$ժամանակ: Ակնհայտ է, որ ընդհանուր առմամբ կան n կենտրոնային իրադարձություններ: Ինչ վերաբերում է շրջանագծի իրադարձություններին, նշենք, որ յուրաքանչյուր նման մշակված իրադարձություն սահմանում է 𝑉𝑜𝑟(𝑃) գագաթ: Հիշեցնենք, որ կեղծ ահազանգերը մշակելուց առաջ հանվում են 𝑄-ից: Դրանք ստեղծվում և ջնջվում են մեկ այլ՝ իրական իրադարձության մշակման ժամանակ, և դրանց վրա ծախսված ժամանակը հաշվի է առնվում այս իրադարձության մշակման ժամանակում։ Հետևաբար, մշակված շրջանագծի իրադարձությունների թիվը չի գերազանցում $2n - 5$-ը: Ժամանակի և հիշողության չափի գնահատումները ապացուցված են:

Մինչ այս կետի վերջնական արդյունքը նշելը, հարկ է մի քանի խոսք ասել վերասեռված դեպքերի մասին։

Ալգորիթմը մշակում է իրադարձությունները վերևից ներքև, ուստի այն դեպքերը, երբ երկու կամ ավելի իրադարձություններ տեղի են ունենում նույն հորիզոնական գծի վրա, վերասեռված են: Դա տեղի է ունենում, օրինակ, երբ մի քանի կենտրոններ ունեն նույն 𝑦-կոորդինատները: Նման իրադարձությունները կարող են մշակվել ցանկացած հաջորդականությամբ, քանի դեռ դրանց 𝑥կոորդինատները տարբեր են, ուստի նման դեպքերում անորոշությունը կարող է կամայականորեն լուծվել: Բայց եթե դա տեղի է ունենում ալգորիթմի հենց սկզբում, այսինքն, եթե առաջին և երկրորդ կենտրոնական իրադարձությունների 𝑦 կոորդինատները նույնն են, ապա անհրաժեշտ է հատուկ ծրագիր, քանի որ դեռ ոչ մի աղեղ չկա, որը կգտնվի երկրորդ կենտրոնից վերև: Հիմա ենթադրենք, որ իրադարձությունների կետերը համընկնում են։ Օրինակ, շրջանագծի մի քանի համընկնող իրադարձություններ տեղի են ունենում, երբ նույն շրջանագծի վրա ընկած են չորս կամ ավելի կենտրոններ և այնպիսին, որ այդ շրջանագծի ներսում կենտրոն չկա: Այս շրջանագծի կենտրոնը Վորոնովի դիագրամի գագաթն է՝ որի աստիճանը առնվազն 4 է: Նման վերասեռված դեպքերը լուծելու համար հնարավոր կլիներ գրել հատուկ ծրագիր, բայց դա ամենևին էլ անհրաժեշտ չէ: Ի՞նչ կլինի, եթե թույլ տանք, որ ալգորիթմը վերամշակի նման իրադարձությունները կամայական կարգով: 4-րդ աստիճանի գագաթ առաջացնելու փոխարեն, այն պարզապես նույն կետում կառաջացնի 3 աստիճանի երկու գագաթ և նրանց միջև զրոյական երկարությամբ կող: Ցանկության դեպքում, նման վերասեռված կողերը կարող են հեռացվել հետմշակման փուլում:

Բացի իրադարձությունների մշակման կարգի ընտրության հետ կապված վերասեռվածությունից, մենք պետք է հաշվի առնենք նաև իրադարձությունների մշակման վերասեռվածությունը: Դա տեղի է ունենում, երբ մշակվող 𝑝~𝑖~ կենտրոնը գտնվում է կոտրվածքների գծի կոտրման կետի տակ: Այս դեպքում ալգորիթմը բաժանում է երկու հարևան աղեղներից մեկը և ստացված կտորների միջև տեղադրում 𝑝~𝑖~ - ին համապատասխան աղեղը, որոնցից մեկի երկարությունը զրոյական է։ Այս զրոյական երկարությամբ հատվածն այժմ եռյակի միջին աղեղն է, որը սահմանում է շրջանագծի իրադարձությունը: Այս շրջանագծի ամենացածր կետը համընկնում է 𝑝~𝑖~ − ի հետ: Ալգորիթմը ներդրում է այս շրջանագծի իրադարձությունը իրադարձությունների 𝑄 հերթի մեջ, քանի որ կոտրվածքների գծի վրա կան երեք հարևան աղեղներ, որոնք սահմանում են այն: Շրջանագծի այս իրադարձությունը մշակման պահին ճիշտ է ստեղծում Վորոնովի դիագրամի գագաթը, իսկ զրոյական երկարությամբ աղեղը կարելի է ջնջել ավելի ուշ։ Մեկ այլ վերասեռված դեպք տեղի է ունենում, երբ երեք հարակից աղեղները կոտրվածքների գծի վրա սահմանվում են երեք համագիծ կենտրոններով: Այս կենտրոնները չեն սահմանում ոչ շրջանագիծ, ոչ էլ շրջանագծի իրադարձություն:

# ԳԼՈՒԽ 2․ ՎՈՐՈՆՈՎԻ ԴԻԱԳՐԱՄԻ ԿԻՐԱՌՈՒԹՅՈՒՆԸ

Վորոնովի դիագրամի ընդհանրացումներից է բարձր կարգի Վորոնովի դիագրամը։ Դիցուք տրված է $P = \left\{ p_{1},p_{2},\ \ldots,\ p_{n} \right\}$ կետերի վերջավոր բազմություն: Ներմուծենք Վորոնովի բջջի հասկացությունը P բազմության ինչ-որ T ենթաբազմության համար։ Վորոնովի բջիջ $T \subset P$ համար կանվանենք՝

$$Vor(T) = \left\{ s \in E^{2}:\ \forall p_{i}\  \in T,\ \forall p\  \in P\backslash T,\ d\left( p_{i},s \right) \leq d(p,s) \right\}$$

## 2.1. Բարձր կարգի Վորոնովի Դիագրամներ, սահմանում և երկրաչափական հատկություններ {#բարձր-կարգի-վորոնովի-դիագրամներ-սահմանում-և-երկրաչափական-հատկություններ}

Դիցուք $R^{2}$ հարթության մեջ տրված է N նմուշային $S = \{ q_{1},\ q_{2},..,q_{N}\}$ բազմությունը, որտեղ յուրաքանչյուր $q_{i}$ կետ ներկայացված է, որպես կարգավորված զույգ՝ ($x_{i},\ y_{i}$), որտեղ $i = 1,\ 2,\ \ldots N$: Երկու՝ $q_{i}$ և $q_{j}$ կետերի միջև հեռավորությունը Lₚ մետրիկայում սահմանվում է $d_{p}\left( q_{i},\ {\ q}_{j} \right) = \left( \left| x_{i} - x_{j} \right|^{p} + \left| y_{i} - y_{j} \right|^{p} \right)^{\frac{1}{p}}\ $, իսկ $d_{\infty} = \left( q_{i},\ q_{j} \right) = max(|x_{i} - x_{j}|,\ |y_{i} - y_{j}|)$։ Մենք կդիտարկենք միայն Էվկլիդեսյան մետրիկան, այսինքն՝ L₂ մետրիկան:

Սովորական Վորոնովի դիագրամում յուրաքանչյուր բազմանկյուն ասոցացվում է մեկ կետի հետ, այսինքն՝ $V(i)$-ն ասոցացվում է $q_{i}$-ի հետ: Այժմ մենք ընդլայնում ենք այս հասկացությունը՝ դիտարկելով կետերի $H$ ենթաբազմության հետ ասոցացված $V(H)$ Վորոնովի բազմանկյունը: Այսինքն՝ $V(H)$-ն այն կետերի երկրաչափական տեղն է, որոնք ավելի մոտ են $H$-ի բոլոր կետերին, քան $H$-ին չպատկանող ցանկացած այլ կետի: Կամ, համարժեքորեն, $V(H) = \bigcap_{q_{i} \in H,\ q_{j} \in S - H}^{}{h(q_{i},q_{j})},$որտեղ$S$-ը նշանակում է կետերի ամբողջական բազմությունը: Պարզության համար կօգտագործենք$\bigcap_{}^{}{h(H,\ S - H)}$՝ նշանակելու համար $\bigcap_{q_{i} \in H,\ q_{j} \in S - H}^{}{h(q_{i},q_{j})}$- ը: Քանի որ $H$-ը կարող է լինել $S$-ի $2^{N}$ հնարավոր ենթաբազմություններից ցանկացածը, ապա $V(H)$-ը, բնականաբար, կարող է և գոյություն չունենալ: Մենք կսահմանափակվենք այն դեպքով, երբ դիտարկվող $H$ ենթաբազմության տարրերի քանակը k է: Մենք սահմանում ենք k-րդ կարգի Վորոնովի դիագրամը, որը նշանակվում է $V_{k}(S)$-ով, որպես $S$-ի բոլոր k-ենթաբազմությունների Վորոնովի բազմանկյունների հավաքածու, այն է՝ $V_{k}(S) = \cup \left\{ V(H):H \subset S,\ |H| = k \right\}$ \[4\]: Քանի որ $V(H)$ բազմանկյուններից մի քանիսը կարող են դատարկ լինել, $V_{k}(S)$-ում բազմանկյունների ընդհանուր քանակը կախված է տրված $N$ կետերի դիրքերից: Ինչպես ցույց կտանք ստորև, այդ թիվը $O(k(N - k))$ է: Նախքան առաջ շարժվելը, մենք կկատարենք հետևյալ դիտարկումները, որոնք հանդիսանում են $V_{1}(S)$-ից $V_{k}(S)$ իտերատիվ կառուցման հիմքը:

Մենք ենթադրում ենք, որ տրված $S\ $բազմության 3-ից ավելի կետեր նույն շրջանագծի վրա չեն գտնվում:

Ենթադրենք, որ $V_{1}(S)$-ը արդեն որոշված է: Թեստային կետը դիագրամում տեղորոշելուց հետո մենք կարող ենք պարզել նրա ամենամոտ հարևանին: Երկրորդ ամենամոտ հարևանը կարող է լինել միայն այն կետերից մեկը, որոնց հետ ասոցացված Վորոնովի բազմանկյունները հարակից են այն բազմանկյանը, որում գտնվում է թեստային կետը: Այսպիսով, եթե մենք կարողանանք որևէ կերպ «տրոհել» այն Վորոնովի բազմանկյունը, որտեղ գտնվում է թեստային կետը, մի շարք ենթատիրույթների, որոնցից յուրաքանչյուրը հարակից է իրեն շրջապատող Վորոնովի բազմանկյուններից մեկին, ապա ենթատիրույթներում թեստային կետի դիրքը կպարզի նրա երկրորդ ամենամոտ հարևանին:

Եթե մենք «տրոհում» իրականացնենք $V_{1}(S)$-ի յուրաքանչյուր Վորոնովի բազմանկյան վրա, արդյունքում կստանանք $V_{2}(S)$-ը: Նմանապես, տրոհելով $V_{2}(S)$-ը, մենք կստանանք $V_{3}(S)$-ը, և այդպես շարունակ: Այսպիսով, այս ընթացակարգի k-1 իտերացիաների արդյունքում մենք կարող ենք $V_{1}(S)$-ից ստանալ k-րդ կարգի Վորոնովի դիագրամը՝ $V_{k}(S)$: Տրված կետի k ամենամոտ հարևանները կարելի է որոշել՝ պարզապես տեղորոշելով թեստային կետը $V_{k}(S)$-ում: Նշենք, որ վերջնական $V_{k}(S)$-ը օգտակար է միայն ֆիքսված k-ի դեպքում, որը հայտնի է նախապես:

Այժմ դիտարկենք հետևյալ օրինակը։ Նկար 2.3-ում ցույց է տրված ութ կետերի վրա կառուցված 1-ին կարգի Վորոնովի դիագրամ: Ենթադրենք, որ q թեստային կետը գտնվում է $V(5$)-ում: Դրա ամենամոտ հարևանը $q_{5}$-ն է: Նրա երկրորդ ամենամոտ հարևանին գտնելու համար մենք կարող ենք արհեստականորեն «ջնջել» $q_{5}$-ը և լուծել ամենամոտ հարևանի խնդիրն այնպես, կարծես կա ընդամենը յոթ կետ: Այդ դեպքում մնացած յոթ կետերի վրա կառուցված Վորոնովի դիագրամը կբաժանի $V(5)$ բազմանկյունը վեց ենթատիրույթների, ինչպես ցույց է տրված Նկար 2․3-ի հաստ գծերով, և այն ենթատիրույթին ասոցացված կետը, որտեղ գտնվում է q-ն, հանդիսանում է երկրորդ ամենամոտ հարևանը: Նկար 2.4-ը ցույց է տալիս$\ V_{2}(S)$-ի վերջնական կառուցված, երբ ձևափոխությունն իրականացվում է բոլոր ութ բազմանկյունների վրա: Նկար 2.5-ում պատկերված է $V_{3}(S)$-ը կետերի նույն բազմության վրա:

Եթե մենք վերադրենք $V_{2}(S)$-ը, իսկ այնուհետև $V_{3}(S)$-ը սկզբնական $V_{1}(S)$ դիագրամի վրա, որոնք համապատասխանաբար ցույց են տրված Նկար 2.6-ում և Նկար 2.7-ում, կտեսնենք, որ ընդհանրացված Վորոնովի դիագրամում հիմնականում առկա է երկու տիպի բազմանկյուն:

Մյուս կողմից, $V_{3}$-ում (Նկար 2.7) կան երկու տիպի տիրույթներ, որոնցից յուրաքանչյուրն ասոցացված է 3 ինդեքսների հետ: Օրինակ՝ $V(1,\ 3,\ 5)$-ը I տիպի է և պարունակում է նոր կող, իսկ V(3, 4, 5)-ը II տիպի է և պարունակում է Վորոնովի կետ, որը գոյություն ուներ $V_{1}$-ում և $V_{2}$-ում:

Այժմ ապացուցենք $k$-րդ կարգի Վորոնովի դիագրամների որոշ հատկություններ: Նախ ներմուծենք որոշ նշանակումներ:

$I = \{ 1,2,\cdots,n\}$ ինդեքսների բազմության վրա $\oplus$ բինար ասոցիատիվ գործողությունը սահմանվում է հետևյալ կերպ: Դիցուք $A$-ն և $B$-ն $I$-ի երկու ենթաբազմություններ են: $A \oplus B$-ն սահմանվում է որպես $\left( A - B) \cup (B - A \right)$, որտեղ "$-$"-ը բազմությունների տարբերությունն է: ($A \oplus B$-ն սովորաբար անվանում են $A$ և $B$ բազմությունների սիմետրիկ տարբերություն:) Հարմարության համար $q_{i}$ կնշանակենք պարզապես $i$-ով: Այսպիսով, $\mathcal{V}\left( q_{i_{1}},q_{i_{2}},\cdots,q_{i_{k}} \right)\ $Վորոնովի բազմանկյունը կնշանակվի $\mathcal{V(}i_{1},i_{2},\cdots,i_{k})$: Քանի որ դիագրամի կողերը երկու կետերի միջնուղղահայացների մասեր են, $\bar{B}(i,j)$-ով կնշանակենք $B(i,j)$ միջնուղղահայացի այն հատվածը, որը հանդիսանում է կող: Ասում ենք, որ $(i,j)$ ինդեքսների զույգը ասոցացված է $\bar{B}(i,j)$ կողի հետ: Դիցուք $\mathcal{V}_{k}(H,I)$-ն նշանակում է $V_{k}(I)$ Վորոնովի դիագրամի $\mathcal{V}_{k}(H)$ բազմանկյունը: Հետագայում մենք $I = \{ 1,\ 2,\ \ldots,\ n\}$բազմությունը կօգտագործենք կետերի $S = \{ q_{1},q_{2},\cdots,q_{n}\}$ բազմությանը համարժեք։

**Լեմմա 2.1․***Տրված է կետերի բազմություն և I բազմության* $H = \{ i_{1},\ i_{2},\ \ldots,i_{k}\}$ *ենթաբազմությունը, որտեղ* $k > 1$*: Եթե *$H_{s} = H - \{ j_{s}\}$*, որտեղ* $j_{s} \in H,s = 1,2,\cdots,m$ *և* $m\  \geq \ k$

*արժեքների համար, H-ի միակ ենթաբազմություններն են, որոնց համար գոյություն ունի  *$\mathcal{V}_{k - 1}(H_{s},I)$*, ապա *$\mathcal{V}_{k}(H,I)$*-ն կարելի է ներկայացնել որպես․*

$$\cup_{s = 1}^{m}(\mathcal{V}_{k - 1}(H_{s},I) \cap \mathcal{V}_{1}(j_{s},I - H_{s}))$$

**Ապացույց.** Դիտարկենք $k - 1$ կարգի $V_{k - 1}(I)$ դիագրամը և $\mathcal{V}_{k}(H,I)$-ի ու $V_{k - 1}(I)$-ի հատումը: Նկատենք, որ $\mathcal{V}_{k}(H,I) \cap \mathcal{V}_{k - 1}(H^{'},I) = \varnothing$, եթե $H^{'} \neq H_{s}$ որևէ $s = 1,2,\cdots,m$-ի համար, հակառակ դեպքում մենք կարող ենք հատման մեջ գտնել այնպիսի $q$ կետ, որի $\left( k-1 \right)$ ամենամոտ հարևանների բազմությունը $q$-ի $k$ ամենամոտ հարևանների բազմության խիստ ենթաբազմություն չէ, ինչն անհնար է: Հետևաբար, ունենք՝

$$\mathcal{V}_{k}(H,I) = \mathcal{V}_{k}(H,I) \cap (\cup_{s = 1}^{m}\mathcal{V}_{k - 1}(H_{s},I)) = \cup_{s = 1}^{m}(\mathcal{V}_{k}(H,I) \cap \mathcal{V}_{k - 1}(H_{s},I))$$

Վերհիշենք, որ $\mathcal{V}_{k}(H,I) = \  \cap h(H,I - H)$։

Մենք ունենք՝

$$\mathcal{V}_{k}(H,I) \cap \mathcal{V}_{k - 1}(H_{s},I) = ( \cap h(H,I - H)) \cap ( \cap h(H_{s},I - H_{s}))$$

$$= ( \cap h(j_{s},I - H) \cap ( \cap h(H_{s},I - H))) \cap ( \cap h(H_{s},I - H_{s}))$$

$$= ( \cap h(j_{s},I - H)) \cap ( \cap h(H_{s},I - H) \cap ( \cap h(H_{s},I - H_{s})))$$

$$= \mathcal{V}_{1}(j_{s},I - H_{s}) \cap ( \cap h(H_{s},I - H) \cap ( \cap h(H_{s},I - H_{s})))$$

Քանի որ

$$\cap h\left( H_{s},I - H_{s} \right) = \  \cap h\left( H_{s},I - H \right) \cap \left( \cap h\left( H_{s},j_{s} \right) \right) \subseteq \  \cap h\left( H_{s},I - H \right),$$

ապա

$\cap h\left( H_{s},I - H \right) \cap \left( \cap h\left( H_{s},I - H_{s} \right) \right) = \  \cap h(H_{s},I - H_{s}) = \mathcal{V}_{k - 1}(H_{s},I)$։

Հետևաբար կստանանք՝

$\mathcal{V}_{k}(H,I) = \cup_{s = 1}^{m}(\mathcal{V}_{k - 1}(H_{s},I) \cap \mathcal{V}_{1}(j_{s},I - H_{s}))$:

**Լեմմա 2.2.** *Տրված է *$k$*-րդ կարգի *$V_{k}(I)$* Վորոնովի դիագրամը: Եթե* $q$*-ն *$V_{k}(I)$*-ի* $\bar{B}(i,j)$* կողի կետ է, ապա *$q$*-ի *$k$*-րդ ամենամոտ հարևանը կամ *$i$*-ն է, կամ *$j$*-ն:*

**Ապացույց.** Այն ակնհայտ է, երբ $k = 1$։ Որպես ինդուկցիայի քայլ՝ դիտարկենք ցանկացած $\mathcal{V}_{k - 1}(H,I)$ բազմանկյուն և $V_{1}(I - H)$ դիագրամը, այսինքն՝ ամենամոտ հարևանի դիագրամը, որտեղից հեռացված է կետերի $H$ բազմությունը։ Դիցուք $q$-ն $\mathcal{V}_{k - 1}(H,I) \cap V_{1}(I - H)$ հատման կետ է այնպես, որ $q$-ն ընկած է $V_{1}(I - H)$-ի $\bar{B}(i,j)$ կողի վրա։ Այսինքն՝ $q \in \mathcal{V}_{1}(i,I - H)$ և $q \in \mathcal{V}_{1}(j,I - H)$։ Համաձայն Լեմմա 2.1-ի$q \in \mathcal{V}_{k}(H \cup \{ i\},I)$ և $q \in \mathcal{V}_{k}(H \cup \{ j\},I)$, ինչը նշանակում է, որ $q$-ն գտնվում է $V_{k}(I)$-ի $\bar{B}(i,j)$-ի վրա։ Քանի որ $q \in \mathcal{V}_{k - 1}(H,I)$, $H$-ը $q$-ի $\left( k-1 \right)$ ամենամոտ հարևանների բազմությունն է։ Ավելին, $q$-ի ամենամոտ հարևանը կետերի $I - H$ բազմությունում կամ $i$-ն է, կամ $j$-ն, որտեղից հետևում է, որ $q$-ի $k$-րդ ամենամոտ հարևանը կամ $i$-ն է, կամ $j$-ն։ 

**Լեմմա 2.3.** *Տրված է *$k$*-րդ կարգի *$V_{k}(I)$* Վորոնովի դիագրամը: *$q$*-ն *$V_{k}(I)$*-ի *$\bar{B}(i,j)$* կողի կետ է այն և միայն այն դեպքում, երբ *$q$* կենտրոնով և *$d(q,i) = d(q,j)$* շառավղով շրջանն իր ներսում պարունակում է *$k - 1$* կետ:*

**Ապացույց.** Եթե $q$-ն $V_{k}(I)$-ի $\bar{B}(i,j)$-ի կետ է, ապա պնդումը ճիշտ է համաձայն Լեմմա 2-ի: Հակառակն ապացուցելու համար դիտարկենք $q$-ի $N(q,\delta)$ շրջակայքը, այսինքն՝ $q$ կենտրոնով և $\delta$ շառավղով շրջան: Քանի որ $q \in B(i,j)$ և գոյություն ունի $q$ կենտրոնով և $d(q,i) = d(q,j)$ շառավղով $K$ շրջան, որը պարունակում է $k - 1$ կետ, հեռավորության մետրիկայի անընդհատությունից ելնելով, մենք կարող ենք $N(q,\delta)$-ում գտնել $q^{'}$ կետ և $q^{'}$ կենտրոնով ու $d(q^{'},i) = d(q^{'},j)$ շառավղով $K^{'}$ շրջան այնպես, որ $K^{'}$-ը նույնպես պարունակի նույն $k - 1$ կետերը: Դիցուք $H$-ը նշանակում է $K$-ում (և $K^{'}$-ում) պարունակվող $k - 1$ կետերի բազմությունը: Քանի որ $q^{'} \in B(i,j)$-ն ցանկացած կետ է $N(q,\delta)$-ում, ապա գոյություն ունի $B(i,j)$-ի մի հատված, որն ընկած է $N(q,\delta)$-ում, այնպես, որ $B(i,j)$-ի այդ հատվածի ցանկացած $u$ կետի համար $u$-ի $k$ ամենամոտ հարևանների բազմությունը $H \cup \{ i\}$ կամ $H \cup \{ j\}$ է: Սա նշանակում է, որ $B(i,j)$-ի այդ հատվածը պետք է ընդհանուր լինի $\mathcal{V}_{k}(H \cup \{ i\})$-ի և $\mathcal{V}_{k}(H \cup \{ j\})$-ի համար $V_{k}(I)$-ում:

Այսպիսով, $q$-ն $V_{k}(I)$-ի $\bar{B}(i,j)$ կողի կետ է: 

**Լեմա 2.4.** *Տրված է n կետերից բաղկացած բազմությունը n-1 կարգի* $V_{n - 1}(I)$ *Վորոնովի դիագրամի n-1 կողերը կազմում են ծառ, այսինքն՝ կողերի վրա գտնվող ցանկացած երկու կետերի համար գոյություն ունի դրանք միացնող միակ ճանապարհ։*

**Ապացույց.** Այն բխում է այն փաստից, որ $V_{n - 1}(I)$-ը ամենահեռու կետի Վորոնովի դիագրամն է, և որ ամենահեռու կետերի Վորոնովի դիագրամի բոլոր բազմանկյուններն անսահմանափակ են \[4\]:

**Լեմմա 2.5.** *Դիցուք *$\mathcal{V}_{k}(H)$*-ը *$u_{1},u_{2},\cdots,u_{i}$* գագաթներով Վորոնովի սահմանափակ բազմանկյուն է, որտեղ *$i \geq 3$*: Եթե բոլոր *$u_{1},u_{2},\cdots,u_{i}$* Վորոնովի կետերը նոր են, ապա *$k$*-ն պետք է լինի *$1$*: Եթե *$k > 1$*, ապա այս *$i$* Վորոնովի կետերից առնվազն 2-ը պետք է լինեն հին:*

**Ապացույց.** Ենթադրենք, որ $\mathcal{V}_{k}(H_{1}),\mathcal{V}_{k}(H_{2}),\cdots,\mathcal{V}_{k}(H_{i})$ բազմանկյունները հարակից են $\mathcal{V}_{k}(H)$-ին այնպես, որ $u_{j}$-ն ընդհանուր է $\mathcal{V}_{k}(H_{j}),\mathcal{V}_{k}(H_{j + 1})$ և $\mathcal{V}_{k}(H)$-ի համար $j = 1,2,\cdots,i - 1$ դեպքերում, իսկ $u_{i}$-ն ընդհանուր է $\mathcal{V}_{k}(H_{i}),\mathcal{V}_{k}(H_{1})$ և $\mathcal{V}_{k}(H)$-ի համար:

1)  Ենթադրենք, որ $u_{1},u_{2},\cdots,u_{i}$-ն նոր են և $k \neq 1$: Քանի որ $H$-ը և $H_{j}$-ն կարող են ներկայացվել, որպես $H = H_{0} \cup \left\{ a \right\},\ \ H_{j} = H_{0} \cup \left\{ a_{j} \right\},j = 1,2,\cdots,i,$ $\mid H_{0} \mid = k - 1$,$\ $ իսկ  $\bar{B}\left( a,a_{j} \right)$ $j = 1,2,\cdots,i$ հանդիսանում են $\mathcal{V}_{k}(H)$-ի կողեր, ապա $a$ կետը պետք է գտնվի $\mathcal{V}_{k}(H)$-ի ներսում: Դիտարկենք $\mathcal{V}_{k}(H)$-ի կողի վրա գտնվող $q$ կետը: Դիցուք $q^{'}$-ը $q$-ով և $a$-ով անցնող ուղղի և $\mathcal{V}_{k}(H)$-ի եզրագծի հատման կետն է: Դիցուք $x$-ը ցանկացած կետ է $H_{0}$-ում: Քանի որ $a$ կետը, ըստ Լեմմա 2.2-ի, $q$-ի և $q^{'}$-ի $k$-րդ ամենամոտ հարևանն է, ապա $d(q,x) < d(q,a)$ և $d(q^{'},x) < d(q^{'},a)$։

Այսպիսով,

$d(q,x) + d\left( q^{'},x \right) < d(q,a) + d(q^{'},a) = d(q,q^{'})$,

ինչը հակասում է եռանկյան անհավասարությանը

$d(q,x) + d(q^{'},x) \geq d(q,q^{'})$:

2)  Ենթադրենք, որ $k > 1$ և միայն մեկը, օրինակ՝ $u_{i}$-ն է հին Վորոնովի կետ: Քանի որ $u_{1},u_{2},\cdots,u_{i - 1}$-ը նոր են, ապա ունենք $H = H_{0} \cup \{ a\},H_{j} = H_{0} \cup \{ a_{j}\},j = 1,2,\cdots,i$: Բայց սա ենթադրում է, որ $u_{i}$-ն, որն ընդհանուր է $H,H_{1}$ և $H_{i}$-ի համար, պետք է լինի նոր, ինչը հակասություն է: 

Այս լեմման ենթադրում է, որ $k > 1$ կարգի Վորոնովի դիագրամի ոչ մի նոր կետ երբեք չի կազմում Վորոնովի բազմանկյուն: Համանման դատողություններով ունենք հետևյալը.

**Հետևանք 2.** *Ոչ մի սահմանափակ *$k$*-րդ կարգի Վորոնովի բազմանկյուն չունի գագաթներ, որոնք բոլորը հին են:*

**Թեորեմ 2.1.** $V_{i + 1}(S)$* Վորոնովի դիագրամը կարելի է ստանալ *$V_{i}(S)$*-ից և *$V_{1}(S - H)$*-ից՝ վերցնելով *$\mathcal{V}_{i}(H)$*-ի և *$V_{1}(S - H)$*-ի հատումները բոլոր այն *$H \subseteq S$* բազմությունների համար, որոնց դեպքում *$\mathcal{V}_{i}(H)$*-ը դատարկ չէ:*

**Լեմմա 2.6.** *Եթե *$\mathcal{V}_{i}(H)$*-ն իր ներսում պարունակում է *$V_{i - 1}(S)$*-ի *$m$* հին Վորոնովի կետեր, ապա այն պարունակում է նաև *$V_{i - 1}(S)$*-ի *$2m + 1$* հին կողեր:*

**Ապացույց.** Դիտարկենք $\mathcal{V}_{i}(H)$-ի և $V_{i - 1}(S)$-ի հատումը: Համաձայն Լեմմա 2.1-ի. 

$$\mathcal{V}_{i}(H,S) = \cup_{s = 1}^{t}(\mathcal{V}_{i - 1}(H_{s},S) \cap \mathcal{V}_{1}(j_{s},S - H_{s}))$$

որտեղ $H_{s} = H - \left\{ j_{s} \right\}$ և $j_{s} \in H$ բոլոր $s = 1,2,\cdots,t$ արժեքների համար: Սա նշանակում է, որ $\mathcal{V}_{i}(H,S)$-ը $t$ տիրույթների միավորում է, որտեղ յուրաքանչյուր տիրույթն իրենից ներկայացնում է այն կետերի երկրաչափական տեղերի հատումը,որոնց $i - 1\ $ամենամոտ հարևանները $H_{s}$-ի կետերն են, և այն կետերի երկրաչափական տեղը, որոնց $i$-րդ ամենամոտ հարևանը $j_{s}$-ն է: Այլ կերպ ասած, յուրաքանչյուր տիրույթ այն կետերի երկրաչափական տեղն է, որոնց $i$ ամենամոտ հարևանները գտնվում են $H_{s} \cup \left\{ j_{s} \right\} = H$ բազմության մեջ, իսկ $i$-րդ ամենամոտ հարևանը $j_{s}$-ն է: Այժմ դիտարկենք $V_{i - 1}(H)$ դիագրամը: Քանի որ $V_{i - 1}(H)$-ը $\mathcal{V}_{i}(H,S)$ բազմանկյունը տրոհում է ճշգրտորեն այն տիրույթների, որոնցից յուրաքանչյուր $R_{j_{s}}$-ը այն կետերի երկրաչափական տեղն է, որոնց $i$ ամենամոտ հարևանները $H$-ի կետերն են, իսկ $i$-րդ ամենամոտ հարևանը $j_{s}$-ն է, ապա յուրաքանչյուր $R_{j_{s}}$ տիրույթ նույնական է $\mathcal{V}_{i - 1}\left( H_{s},S \right) \cap \mathcal{V}_{1}\left( j_{s},S - H_{s} \right)$-ին: Համաձայն Լեմմա 2.4-ի՝ $\mathcal{V}_{i}(H,S)$-ը տրոհող կողերը կազմում են ծառ: Հետևաբար, եթե կա $m$ կետ, ապա գոյություն ունի $2m + 1$ կող:

**Լեմմա 2.7*.**** *$V_{k}(S)$*-ում նոր Վորոնովի կետերի քանակն է՝*

$$I_{k}^{'} = 2k(N - 1) - k(k - 1) - \sum_{i = 1}^{k}\mathcal{S}_{i}.$$

## 2.2. k-րդ կարգի Վորոնովի դիագրամի կառուցման ալգորիթմը {#k-րդ-կարգի-վորոնովի-դիագրամի-կառուցման-ալգորիթմը}

Այս բաժնում մենք կներկայացնենք $1$-ին կարգի $V_{1}(S)$ Վորոնովի դիագրամի մեկ առանձին Վորոնովի բազմանկյան, դիցուք՝ $\mathcal{V(}q_{m})$-ի տրոհման ալգորիթմը: Ենթադրենք, որ $\mathcal{V(}q_{0}\mathcal{),\cdots,V(}q_{n - 1})$ բազմանկյունները հարակից են $\mathcal{V(}q_{m})$ բազմանկյանը: Մենք ցանկանում ենք $\mathcal{V(}q_{m})$-ը տրոհել $n$ ենթատիրույթների այնպես, որ յուրաքանչյուր $r_{i}$ ենթատիրույթ լինի այն կետերի երկրաչափական տեղը, որոնք ավելի մոտ են $q_{i}$-ին, քան ցանկացած այլ կետի, բացառությամբ $q_{m}$-ի, որտեղ $i = 0,1,\cdots,n - 1$ (տե՛ս Թեորեմ 1): Ինչպես նշվեց 2.1 ենթագլխում, $\mathcal{V(}q_{m})$-ի տրոհումն ըստ էության համարժեք է $q_{m}$-ի արհեստականորեն հեռացմանը և մնացած $q_{0},q_{1},\cdots,q_{n - 1}$ կետերի համար Վորոնովի դիագրամի կառուցմանը: $q_{m}$-ի հեռացման հետևանքը բազմանկյան եզրագծի վերացումն է և $\mathcal{V(}q_{m})$-ի գագաթներում հատվող կողերի շարունակումը դեպի $\mathcal{V(}q_{m})$-ի ներս, ինչի արդյունքում $\mathcal{V(}q_{m})$ բազմանկյունը տրոհվում է: Դիցուք $\mathcal{V(}q_{m})$-ի գագաթները նշանակված են որպես $I_{0},I_{1},\cdots,I_{n - 1}$ և պահվում են որպես կապակցված ցուցակ։ Բազմանկյան յուրաքանչյուր $(I_{i},\ I_{i + 1})$ կող հանդիսանում է $B(q_{i},q_{m})$ միջնուղղահայացի մի հատված և ներկայացվում է $(i,m)$ ինդեքսների զույգով: Ըստ ենթադրության՝ յուրաքանչյուր $I_{i}$ գագաթ երեք կողերի հատման կետ է, որոնք ներկայացվում են $\left( i,m),(i - 1,m \right)$ և $(i,i - 1)$ զույգերով: Նկատեք, որ երեք կողերի ինդեքսների զույգերը բավարարում են $(a,b) \oplus (b,c) \oplus (c,a) = \varnothing$ հատկությանը: Դիցուք $\text{IND}(I_{i})$-ով նշանակում ենք այն կողը, որը հարակից է $I_{i}$-ին և չի գտնվում $\mathcal{V(}q_{m})$ բազմանկյան եզրագծի վրա: Եթե երկու եզրային կողերը, որոնք հարակից են $I_{i}$-ին, ներկայացվում են $(a,b)$-ով և $(a,\ c)$-ով, ապա $\text{IND}(I_{i}) = (a,b) \oplus (a,c) = (b,c)$: Նկ. 2.8-ը ցույց է տալիս տիպիկ $\mathcal{V}(10)$ Վորոնովի բազմանկյուն, որը պետք է տրոհվի:

Նկատեք, որ Լեմմա 2.5-ի արդյունքը վճռորոշ է միաձուլման ալգորիթմի ճշտության

համար, քանի որ այն ապահովում է, որ միաձուլման գործընթացում «միաձուլման գծերը» երբեք ցիկլեր չկազմեն:

Ենթադրենք, որ մենք ստացել ենք $i$-րդ կարգի $V_{i}(S)$ Վորոնովի դիագրամը, որտեղ $k > i \geq 1$: Վերհիշենք, որ կան երկու տիպի Վորոնովի կետեր՝ հին կետեր, որոնք գոյություն ունեին $V_{i - 1}(S)$-ում, և նոր կետեր, որոնք նոր են ստեղծվել: Հիմնվելով նախորդ քննարկումների վրա՝ միայն Վորոնովի նոր կետերի բազմությունն է անհրաժեշտ $V_{i + 1}(S)$ դիագրամը կառուցելու համար: Ինդուկցիայի սկզբունքով ենթադրենք, որ յուրաքանչյուր $\mathcal{V}_{i}(H_{j})$ Վորոնովի բազմանկյուն ասոցացված է $i$ կետերի հետ, այսինքն՝ $\mathcal{V}_{i}(H_{j})$-ն այն կետերի երկրաչափական տեղն է, որոնք ավելի մոտ են այդ $i$ կետերին, քան $S$-ի մնացած կետերին, իսկ նրա գագաթները ներկայացված են որպես երկկողմանի կապակցված շրջանաձև ցուցակ: Քանի որ $\mathcal{V}_{i}(H_{j})$-ի գագաթներից մի քանիսը հին Վորոնովի կետեր են, իսկ որոշները՝ նոր Վորոնովի կետեր, միայն այդ նոր Վորոնովի կետերն են լրացուցիչ կերպով կապակցված միմյանց հետ՝ օգտագործելով լրացուցիչ ցուցիչ: Նկատեք, որ $V_{1}(S)$-ի բոլոր գագաթները նոր Վորոնովի կետեր են: Այժմ $\mathcal{V}_{i}(H_{j})$-ն տրոհելու համար մենք նախ ստանում ենք նոր Վորոնովի կետերի բազմությունը, որոնցից յուրաքանչյուրն ասոցացված է $\text{IND}(I_{j})$-ով ներկայացվող կողի հետ, որտեղ $I_{j}$-ն $\mathcal{V}_{i}(H_{j})$-ի նոր Վորոնովի կետ է, այնուհետև կիրառում ենք հենց նոր նկարագրված «բաժանի՛ր և տիրի՛ր» եղանակը: Յուրաքանչյուր Վորոնովի բազմանկյուն տրոհելուց հետո մեզ անհրաժեշտ է միմյանց շղթայել յուրաքանչյուր նոր ստեղծված Վորոնովի բազմանկյան գագաթները, այն ասոցացնել $i + 1$ կետերից բաղկացած նոր բազմության հետ, և միևնույն ժամանակ, երբ մենք ձևավորում ենք յուրաքանչյուր բազմանկյան գագաթների ցուցակը, մենք կապակցում ենք բազմանկյան նոր գագաթները՝ ցուցակ ձևավորելով: Այս եղանակով մենք կարող ենք ստանալ $i + 1$ կարգի $V_{i + 1}(S)$ դիագրամը:

Այժմ վերլուծենք ալգորիթմի աշխատանքի ժամանակը: Ենթադրենք, որ տրոհման ենթակա $\mathcal{V}_{i}(H)$ բազմանկյունն ունի Վորոնովի $s$ նոր կետեր՝ $I_{1},I_{2},\cdots,I_{s}$: $\mathcal{V}_{i}(H)$-ի տրոհման գործընթացը համարժեք է $s$ կետերից բաղկացած բազմության համար 1-ին կարգի Վորոնովի դիագրամի կողերի որոնմանը , որոնց ինդեքսները հանդիսանում են $\text{IND}(I_{i}),i = 1,2,\cdots,s$ կողերին ասոցացված ինդեքսների միավորումը, և հետևաբար այն պահանջում է $O(s\log s)$ ժամանակ: Քանի որ $V_{i}(S)$-ում կան $N_{i}$ Վորոնովի բազմանկյուններ և $O(iN)$ նոր Վորոնովի կետեր (Լեմմա 2.6), անհրաժեշտ գործողությունների ընդհանուր քանակը կազմում է՝

$$\sum_{j = 1}^{N_{i}}{O(}s_{j}\log s_{j}) = O(iN\log N)$$

Հիշողության պահանջվող ծավալը կազմում է $O(N_{i + 1}) = O((i + 1)(N - (i + 1)))$: Քանի որ $k$-րդ կարգի Վորոնովի դիագրամ ստանալու համար պահանջվում է $k - 1$ իտերացիա, ամբողջ աշխատանքը կազմում է $\sum_{i = 1}^{k - 1}{O(iN\log N) = O(}k^{2}N\log N)$: Հիշողության պահանջվող ծավալը $O(k \cdot N_{k})$ է, որը հավասար է $O(k^{2}(N - k))$, քանի որ կան $O(N_{k})$ տիրույթներ, և յուրաքանչյուր տիրույթ ասոցացված է $k$ կետերի հետ:

## 2.3. Վորոնովի դիագրամի կիրառումը փակ տարածքում օբյեկտի դիրքի տեղորոշման համար {#վորոնովի-դիագրամի-կիրառումը-փակ-տարածքում-օբյեկտի-դիրքի-տեղորոշման-համար}

BLE (Bluetooth Low Energy) -- սա անլար կապի տեխնոլոգիա է, որը կիրառում են ոչ մեծ ծավալի տվյալների տեղափոխման համար՝ ծախսելով շատ քիչ էներգիա։ Այս տեխնոլոգիայով տվյալները, որպես կանոն հնարավոր է լինում տեղափոխել 10-30 մետր։ BLE-ն օգտագործում են ֆիտնես շղթաներում, անլար սենսորներում, փակ տարածքում օբյեկտի դիրքի տեղորոշման bluetooth - տվիչներում և բժշկական որոշ սարքավորումներում \[3\]:

RSSI (Received Signal Strength Indicator) - թվային արժեքներով ներկայացվող ցուցանիշ է, որը ցույց է տալիս, թե որքան ուժեղ է ընդունված ազդանշանը։ Սովորաբար այն ներկայացվում է բացասական թվերով (օր․ -30 dBm; -70 dbm): Որքան թիվը ավելի մոտ է 0-ին, այնքան ազդանշանը ուժեղ է։

Դիտարկենք մոտեցում՝ կապված BLE տեխնոլոգիաների կիրառմամբ։ Փակ տարածքում օբյեկտի տեղորոշման համար օգտագործված է ֆիքսված ընդունիչ սարքեր, որոնք ստանում են օբյեկտի հետ նույնացված տվիչի RSSI պարամետրերի մասին ինֆորմացիա։ Տվիչի արձակած ազդանշանների հաճախությունը կարող է փոխվել կախված օգտագործվող սարքավորումից։ Եվս մեկ անգամ նշենք, որ RSSI-ը համարվում է անկայուն և նրա արժեքի վրա կարող է մեծ ազդեցություն ունենալ ցանկացած խոչընդոտ, որն առկա է ազդանշանի տարածման ճանապարհին և նույնիսկ կարող է ազդել տարածքի կառուցվածքը։ Դա կապված է ազդանշանի հզորության շատ արագ մարումով, որը կախված է միջավայրի գործոններից և RSSI-ի արժեքների հեռավորության էքսպոնենցիալ կախվածությունից։

Հետևաբար ընտրվել է համեմատական մոտեցում՝ օգտագործելով RSSI արժեքների համեմատությունը տարբեր ընդունիչների միջև, այլ ոչ թե նրանց բացարձակ արժեքները։ Բոլոր ընդունիչ սարքերից ընտրում ենք 2-3 ամենաբարձր RSSI արժեքներ ունեցողները։ Ընտրված («հուսալի») ընդունիչ սարքերի քանակը կախված է օգտագործվող մոդելի RSSI սահմանային արժեքով, և մոդելի այլ պարամետրերից։ Վստահելի ընդունիչ սարքերի հավաքածուի որոշման համար, իդեալական պայմաններում, կնշանակեր որ նրանցից յուրաքանչյուրի հեռավորությունը տվիչից ավելի փոքր է, քան հեռավորությունը տվիչի մնացած այլ ընդունիչ սարքերից։

![](media/image30.png){width="6.6930555555555555in" height="2.7083333333333335in"}*Նկար 2.11 - Վորոնովի դիագրամի համապատասխան բջջին պատկանելիության որոշում։*

Յուրաքանչյուր ընդունիչ դիտարկենք որպես կետ հարթության վրա։ Դիցուք P-ն բոլոր այդ կետերի բազմությունն է․ այդ կետերի բազմության հիման վրա կարելի է կառուցել 2-րդ կամ 3-րդ կարգի Վորոնովի դիագրամ, կախված հուսալի ընդունիչների քանակից։ Օբյեկտի դիրքը որոշվում է՝ Վորոնովի դիագրամի ըստ այն բջջի, որին այն պատկանում է (տես․ Նկար 2.11)։

Թեստավորման արդյունքները ցույց են տվել, որ նույնիսկ փոքր և քիչ խտությամբ ցանցի դեպքում մեթոդը ապահովում է մոտավորապես 80% ճշգրտություն։ Օբյեկտի դիրքի որոշման սխալի չափը գնահատվում է համապատասխան Վորոնովի բջջի տրամագծով։ Ակնհայտ է, որ ճշգրտությունը կարելի է բարձրացնել՝ մեծացնելով ընդունիչների քանակը և օպտիմալացնելով նրանց փոխադարձ տեղակայումը \[3\]:

Դիցուք տրված են $P_{1},\ P_{2},\ldots,P_{n} \in R^{2}$ տվիչների կոորդինատները և պետք է գնահատել $X = (x,y)$ օբյեկտի դիրքը։ Վորոնովի դիագրամով տիրույթը տրոհվում է մասերի (բջիջների)։ Օբյեկտը բոլոր տվիչներից ընդունում է ազդանշանը։

![](media/image31.png){width="6.347222222222222in" height="2.8120603674540683in"}*Նկար 2.12 - Բջջի կառուցման ճշգրտված տարբերակ։*

Ընտրվում է մեծագույն հզորությամբ ազդանշանով (RSSI) տվիչը։ Այդ դեպքում օբյեկտը գտնվում է այդ տվիչը որպես կենտրոն ունեցող Վորոնովի դիագրամի բիջիջում (ենթադրվում է, որ բոլոր տվիչների հզորությունները նույնն են) ։

Դիտարկենք հետևյալ խնդիրները՝

**Խնդիր 1.** Դիցուք $10 \times 10$ (մ) սենյակում տվիչները գտնվում են $P_{1} = (2,\ 2),\ P_{2} = (8,\ 2),\ \ P_{3} = (5,\ 8)$ կետերում:

Վորոնովի դիագրամը սենյակում է տրոհում է երեք մասի (Նկար 2.13): Օբյեկտի ընդունած ազդանշանների ցուցանիշներն են\`

$${RSSI}_{1} = - 60;\ {RSSI}_{2} = - 80;\ {RSSI}_{3} = - 73$$

Քանի որ $\max{RSSI} = - 60$, ուստի օբյեկտը գտնվում է $P_{1}$ կենտրոնով Վորոնովի բջիջում։

Կոնկրետ կիրառական խնդիրներ լուծելիս հաճախ օգտվում են $RSSI$ լոգարիթմական մոդելից: Դա ազդանշանի մարման մոդել է․

> $RSSI = A - 10*nlgd$ , (2.1)

Դիտարկենք հաջորդ խնդիրը։

**Խնդիր 3:** Դիցուք այժմ նույն $12 \times 8$ սենյակում ունենք $P_{1}(2;\ 2),\ P_{2}(10;\ 2),\ \ P_{3}(6;6),\ $

$P_{4}(2;6)$, տրված է x=5, $0 \leq y \leq 5$ պատ և հայտնի են ${RSSI}_{1} = - 46dBm,\ {RSSI}_{4} = - 53dBm,\ $

${RSSI}_{3} = - 61dBm,\ {RSSI}_{2} = - 63dBm\ $:

Ըստ RSSI-ի տվյալների ունենք․

$P_{1} \prec P_{4} \prec P_{3} \prec P_{2}$,

որտեղ $\prec$ - սիմվոլը նշանակում է, որ «$պետք\ է\ լինի\ մոտ$» ։

Եթե $P_{5}$-ով նշանակենք պատը որոշող հատվածը, ապա $G = \left\{ (x;y) \right|0 \leq x \leq 12;0 \leq y \leq 8\}$ տիրույթում $P_{1},\ \ P_{2},\ \ P_{3},\ \ P_{4},\ \ P_{5}\ $օբյեկտների Վորոնովի 2-րդ կարգի դիագրամի յուրաքանչյուր $V_{\left( P_{k},P_{m}\  \right)}$ -- բջիջ սահմանվում է․

$V_{\left( P_{k},P_{m}\  \right)} = \left\{ M(x;y)\ \epsilon\ G \right|\ \rho\left( M;P_{j} \right) \leq \rho\left( M;P_{i} \right);\ j = m;k;m < k;i \neq j;\ i,j,m,k = 1,2,3,4,5\}$, որտեղ $\rho(M;P)$-ն M կետի հեռավորությունն է P օբյեկտից։

*Նկար 2.15 -- Երկրորդ կարգի Վորոնովի Դիագրամի պատկերումը խոչընդոտների դեպքում։*

Ունենք $A = - 40dBm$, n=2, պատը տալիս է $8dBm$: Ըստ (2.2) բանաձևի հեռավորությունը կհաշվենք\`

$$d = 10^{\frac{A - RSSI - W}{10n}}$$

բանաձևով։

$P_{1}$-ի համար՝ $d_{1} = 10^{\frac{- 40 + 46}{20}} = 1.995մ$,

$P_{4}$-ի համար՝ $d_{2} = 10^{\frac{- 40 + 53}{20}} = 4.47մ$,

$P_{3}$-ի համար՝ $d_{3} = 10^{\frac{- 40 + 61 - 8}{20}} = 4.47մ$,

$P_{2}$-ի համար՝ $d_{4} = 10^{\frac{- 40 + 63 - 8}{20}} = 5.62մ$։

Դիտարկենք $P_{1}$ կենտրոնով և $d_{1}$ շառավղով շրջանագիծը

${(x - 2)}^{2} + {(y - 2)}^{2} = {d_{1}}^{2}$։

$P_{4}$ կենտրոնով և $d_{4}$ շառավղով շրջանագծի համար կունենանք

$${(x - 2)}^{2} + {(y - 6)}^{2} = {d_{4}}^{2}$$

Այդ շրջանագծերի հատումից կստանանք․

$\left\{ \begin{array}{r}
{(x - 2)}^{2} + {(y - 2)}^{2} = {d_{1}}^{2} \\
{(x - 2)}^{2} + {(y - 6)}^{2} = {d_{4}}^{2}
\end{array} \right.\  = > (0;2)\ կամ\ (4;2)$։

Ստուգենք (4; 2) կետը։ $P_{3}$ կետից դրա հեռավորությունը կլինի $\sqrt{{(4 - 6)}^{2} + {(2 - 6)}^{2}} \approx 4.47$ համնկնում է $d_{3}$ հետ։ Տեսնենք (4; 2) և (6; 6) կետերն իրար միացնող ուղիղը հատում է պատը։ Պարզագույն հաշվարկով ուղիղը հատում է (5; 4) կետում, ուստի 8dBm տուգանային արժեքը ճիշտ է վերցնել։ $P_{2}$ կետից հեռավորությունը ՝ $\sqrt{{(4 - 10)}^{2} + {(2 - 2)}^{2}} = 6$, շատ մոտ է $d_{2}$-ին։

(0; 2) կետի դեպքում հեռավորությունը $P_{3}$-ից՝ $\sqrt{36 + 16} \approx 7.2$, բայց պետք է լինի 4.47 ուստի այս դեպքը չի դիտարկվի։

Այսպիսով օբյեկտի հնարավոր դիրքը՝ (4; 2) - ն է։ (նկար 2.16)

![](media/image35.jpg){width="5.125in" height="3.611570428696413in"}

Նկար 2.16 -- Երկրորդ կարգի Վորոնովի դիագրամի պատկերումը նոր օբյեկտի հնարավոր դիրքով։

Բարձր կարգի Վորոնովի դիագրամները հաճախ օգտագործվում են այնպիսի իրավիճակներում, երբ մի քանի դրոններ պետք է համատեղ վերահսկեն կամ սպասարկեն տարածք կամ օբյեկտներ։ Սա կիրառվում է որոնողական գործողություններում, հրդեհների մոնիթորինգում, սահմանների հսկման մեջ և այլն։

Ներկայացնենք կոնկրետ կիրառական խնդիր՝ կորած մարդու որոնում մի քանի դրոններով։ Դիցուք տրված է որոնման տարածք.

$$\Omega \in \ R^{2}$$

oրինակ՝ անտառ կամ լեռնային տարածք։ Կան k հատ որոնող դրոններ․

$D_{1},\ D_{2},\ \ldots,\ D_{k}$:

Պետք է որոնումը կազմակերպել այնպես, որ․

ա) ամբողջ տարածքը ստուգվի հնարավորինս արագ,

բ) դրոնները չկրկնեն նույն աշխատանքը,

գ) որոնումը լինի հավասարաչափ։

Տվյալ պահին կառուցվում է $D_{i}$ դրոնների (որպես $R^{2}$-ի կետերի) երրորդ կարգի Վորոնովի դիագրամ։ Այդ դիագրամի կամայական բջջի յուրաքանչյուր կետ հսկվում է(ստուգվում է) երեք դրոններով։ Այդ պահին պետք է հաշվարկվի յուրաքանչյուր բջջի (որը բազմանկյուն է) կենտրոնը: Եթե այդ բազմանկությունը ունի

$$\left( x_{1},\ y_{1} \right),\ \left( x_{2},\ y_{2} \right),\ \ldots\ \left( x_{n},\ y_{n} \right)$$

գագաթներ, ապա դրա մակերեսը՝

$A = \frac{1}{2}\ \sum_{i = 1}^{n}{(x_{i}y_{i + 1} - x_{i}y_{i + 1})}$,

իսկ կենտրոնը ՝

$$C_{x} = \frac{1}{6A}\sum_{i = 1}^{n}{(x_{i} + x_{i + 1})(x_{i}y_{i + 1} - x_{i}y_{i + 1})}$$

$$C_{y} = \frac{1}{6A}\sum_{i = 1}^{n}{(y_{i} + y_{i + 1})(x_{i}y_{i + 1} - x_{i}y_{i + 1})}$$

(որտեղ $x_{n + 1} = x_{1}$, $y_{n + 1} = y_{1}$):

Յուրաքանչյուր բջիջ համապատասխանում է երեք դրոններին։ Սա նշանակում է որ մի դրոն սովորաբար մասնակցում է մի քանի 3-րդ կարգի բջիջների կառուցվածքում։ Օրինակ՝ $D_{i}$-ն կարող է մասնակցել $V_{3}\left( D_{i},\ D_{j},\ D_{k} \right),\ V_{3}\left( D_{i},\ D_{j},\ D_{l} \right),\ V_{3}(D_{i},\ D_{k},\ D_{l})$ բջիջների կառուցվածքում։ Եթե այդ բջիջների կենտրոնները՝

$C_{ijk},\ C_{ijl},\ C_{jkl}$:

$D_{i}$ դրոնի շարժման նպատակակետը վերցվում է դրանց միջինը․

$$T_{i} = \frac{C_{ijk} + C_{ijl} + \ C_{jkl}}{3}$$

Դրոնի շարժման հետագիծը կտրվի

$\dot{D_{i}}$ $= k(T_{i} - D_{i})$

Բանաձևով, որտեղ $(T_{i} - D_{i})$ վեկտորը ցույց է տալիս ուղղությունը դեպի նպատակակետը, k\>0 արագության գործակից, $\dot{D_{i}}$-դրոնի արագության վեկտորն է, որը ցույց է տալիս, թե ինչ արագությամբ և որ ուղղությամբ պետք է շարժվի դրոնը։ $i$-րդ դրոնի տեղը (դիրքը) ժամանակի t պահին՝ $D_{i}(t)$:

$$D_{i}(t) = T_{i} + (D_{i}(0) - \ T_{i})e^{- kt}$$

Սկզբնական փուլում $k = 0.3$: Այն պահին, երբ դրոններից մեկը (կամ մի քանիսը) հայտնաբերում են փնտրվողին (որն այդ պահին ինչ-որ բջիջում է), ապա այն 3 դրոնները, որոնք պատասխանատու են տվյալ բջիջի համար, կտրուկ մեծացնում են իրենց k-ն (օրինակ՝ k=1.5), իսկ մնացած դրոններից կարող են փոքրացնել իրենց k-ն, որպեսզի խնայեն էներգիան և չխանգարեն հիմնական խմբին։

# ԳԼՈՒԽ 4. ՎՈՐՈՆՈՎԻ ԴԻԱԳՐԱՄԻ ԿԻՐԱՌՈՒՄԸ ՎՏԱՆԳԱՎՈՐ ԳՈՐԾՈՆԻ (ՓՈՇԻ) ԱՂԲՅՈՒՐՆԵՐՈՎ ՓԱԿ ՏԱՐԱԾՔՈՒՄ, ՕԴԱՓՈԽՈՒԹՅԱՆ ՀԱՄԱԿԱՐԳԻ ՕՊՏԻՄԱԼ ԻՐԱԿԱՆԱՑՄԱՆ ՀԱՄԱՐ {#գլուխ-4.-վորոնովի-դիագրամի-կիրառումը-վտանգավոր-գործոնի-փոշի-աղբյուրներով-փակ-տարածքում-օդափոխության-համակարգի-օպտիմալ-իրականացման-համար}

Փակ արտադրական և գրասենյակային տարածքներում վնասակար գործոնների վերահսկումը հանդիսանում է կենսագործունեության անվտանգության հիմնարար խնդիրներից մեկը։ Օդում կախված փոշին դասվում է առավել տարածված և առողջության համար վտանգավոր գործոնների շարքին, քանի որ այն կարող է առաջացնել շնչառական համակարգի խանգարումներ, մասնագիտական հիվանդություններ, ալերգիկ ռեակցիաներ և աշխատունակության նվազում \[1\], \[2\], \[3\]։

Արտադրական միջավայրում օդի որակի ապահովման հիմնական տեխնիկական միջոցներից է արդյունավետ օդափոխության համակարգի նախագծումն ու շահագործումը \[4\], \[5\], \[6\]։ Սակայն բազմաղբյուր աղտոտման պայմաններում օդի հոսքերի ճիշտ կազմակերպումը պահանջում է տարածական վերլուծություն և մաթեմատիկական մոդելավորում։ Այս համատեքստում նպատակահարմար է կիրառել Վորոնովի դիագրամը՝ որպես տարածքի բաժանման և վտանգի գոտիների վերլուծության գործիք։

Փոշին հանդիսանում է պինդ մասնիկների դիսպերս համակարգ, որը կախված վիճակում առկա է օդում։ Այն կարող է լինել անօրգանական (մետաղական, սիլիկատային), օրգանական կամ խառը բնույթի։ Փոշու մասնիկների չափը, քիմիական կազմը և կոնցենտրացիան որոշում են դրա վտանգավորության աստիճանը \[1\], \[2\]։

Փոշու ազդեցության հիմնական հետևանքներն են՝

- շնչառական ուղիների գրգռում,

- թոքերի քրոնիկական հիվանդություններ,

- ֆիբրոգեն ազդեցություն,

- աշխատունակության և արտադրողականության նվազում \[3\], \[5\]։

Փակ տարածքներում աղտոտիչների բաշխումը սովորաբար անհամասեռ է, քանի որ կախված է օդի հոսքերի ուղղությունից, աղբյուրների տեղակայությունից և տարածքի երկրաչափությունից \[4\]։ Հետևաբար անհրաժեշտ է գնահատել ոչ միայն փոշու միջին կոնցենտրացիան, այլև դրա տարածական բաշխումը։

Օդափոխության համակարգի հիմնական նպատակը վնասակար նյութերի հեռացումն է և մաքուր օդի մատակարարումը՝ պահպանելով նորմատիվային թույլատրելի սահմանները \[5\]։

Օդափոխության արդյունավետությունը պայմանավորված է հետևյալ գործոններով․

- օդի հոսքի արագությամբ,

- ներմուծման և արտածման բացվածքների տեղադրմամբ,

- տարածքի բարձրությամբ և ծավալով,

- աղտոտման աղբյուրների դիրքով \[4\]։

Միայն օդափոխության ծավալի մեծացումը բավարար չէ համակարգի արդյունավետությունը բարձրացնելու համար․ անհրաժեշտ է ապահովել օդի հոսքերի ռացիոնալ բաշխում և նվազեցնել «մեռած գոտիների» առաջացումը \[3\], \[5\]։

Այս խնդիրը կարելի է դիտարկել որպես օպտիմալացման խնդիր, որտեղ անհրաժեշտ է գտնել օդափոխիչների տեղադրման այնպիսի սխեմա, որը նվազագույնի կհասցնի վտանգավոր գործոնի տարածական կոնցենտրացիան։

Փակ տարածքում փոշու տարածումը կարելի է գնահատել հեռավորության ֆունկցիայի միջոցով։ Եթե յուրաքանչյուր աղբյուր ունի ինտենսիվություն $w_{i}$, ապա կարելի է սահմանել կշռված հեռավորություն․

$$d_{w}(x,p_{i}) = \frac{d(x,p_{i})}{w_{i}}.$$

Երկրորդ կարգի մոտեցման դեպքում տարածքի յուրաքանչյուր կետի ռիսկը կարելի է գնահատել հետևյալ ֆունկցիայով․

$$R(x) = \frac{1}{d_{1}} + \frac{1}{d_{2}},$$

որտեղ $d_{1}$և $d_{2}$համապատասխանաբար երկու ամենամոտ աղբյուրների հեռավորություններն են։

Բարձր $R(x)$ արժեքները բնորոշ են համակցված ազդեցության գոտիներին։ Օդափոխիչների տեղադրումը նման կետերում նպաստում է առավելագույն աղտոտման հեռացմանը։Այս մոտեցումը հնարավորություն է տալիս․

- բացահայտել բարձր ռիսկային հատվածները,

- նվազեցնել ընդհանուր կոնցենտրացիայի ինտեգրալ արժեքը,

- ապահովել օդի հոսքերի հավասարաչափ բաշխում,

- բարձրացնել աշխատողների անվտանգության մակարդակը \[1\], \[5\]։

Վորոնովի երկրորդ կարգի դիագրամի կիրառումը փոշու աղբյուրներով փակ տարածքներում թույլ է տալիս կատարել տարածական վերլուծություն և ստանալ ինժեներական հիմնավորված լուծումներ օդափոխության նախագծման համար։ Մեթոդը ապահովում է.

- արտադրական միջավայրի սանիտարահիգիենիկ ցուցանիշների բարելավում,

- վտանգավոր գործոնների վերահսկման արդյունավետության բարձրացում,

- էներգախնայող լուծումների ընտրություն,

- նորմատիվ պահանջների պահպանում \[2\], \[4\], \[6\]։

Այսպիսով, մաթեմատիկական մոդելավորման և կենսագործունեության անվտանգության սկզբունքների համադրումը հնարավորություն է տալիս ստեղծել գիտականորեն հիմնավորված օդափոխության օպտիմալ համակարգեր՝ ուղղված աշխատողների առողջության և անվտանգ աշխատանքի պայմանների ապահովմանը։

Երկրորդ կարգի Վորոնովի դիագրամի կառուցումը թույլ տվեց սահմանել այն գոտիները, որտեղ միաժամանակ գերակշռում է երկու փոշու աղբյուրների ազդեցությունը։ Ստացված տարածական քարտեզը ցույց տվեց, որ առավել մեծ համակցված ազդեցություն դիտվում է կենտրոնական հատվածում, որտեղ միաժամանակ ներգործում են հարակից երեք արտադրական հանգույցներ։

Դիտարկվում է 12×10 մ չափերով փակ արտադրական տարածք, որտեղ տեղակայված են 8 կետային փոշու աղբյուրներ՝ տարբեր ինտենսիվություններով.

$$S_{i} = (x_{i},y_{i},w_{i})$$

Որտեղ\` $x_{i},y_{i}$ աղբյուրի կոորդինատներն են, իսկ $w_{i}$ փոշու արտանետման ինտենսիվությունը։

Տրված են հետևյալ աղբյուրները\` (2, 2, 1.2), (10, 2, 1.0),  (6, 6, 1.5), (2, 6, 0.9), (9, 5, 1.3),

(4, 1, 0.8), (8, 7, 1.1), (1, 4, 1.0)։ Նպատակն է՝ գտնել օդափոխության համակարգի արտածման բացվածքի օպտիմալ տեղադրումը, որպեսզի նվազեցվի փոշու համակցված ազդեցությունը շնչառական գոտում։

Նկար 4.1.Երկրորդ կարգի Վորոնովի դիագրամը 8 կետային փոշային աղբյուրների համար։

Նկար 8.1. պատկերված Կանաչ կետերըը փոշու աղբյուրներն են՝ հաստոցներ, սարքավորումներ և այլն։ Գունավոր տարածքները՝ ցույց է տալիս այն տարածքը, որտեղ. տվյալ տարածքի յուրաքանչյուր կետի համար նույն երկու աղբյուրներն են ամենամոտը։ Այսինքն՝ այդ գոտիներում գործում է երկու աղբյուրների համակցված ազդեցություն։

Կառուցված երկրորդ կարգի Վորոնովի դիագրամը ցույց է տալիս փոշու աղբյուրների համակցված ազդեցության տարածական բաշխումը։ Յուրաքանչյուր գունավոր բջիջ համապատասխանում է այն տարածքին, որտեղ երկու կոնկրետ աղբյուրներ հանդիսանում են ամենամոտը և ձևավորում են գերակշռող ազդեցություն։

Դիագրամի կենտրոնական հատվածում դիտվում են փոքր մակերես ունեցող բջիջներ, ինչը վկայում է աղբյուրների խտության բարձր մակարդակի մասին։ Նման հատվածներում համակցված փոշային ազդեցությունը առավել ինտենսիվ է և պահանջում է օդափոխության համակարգի նպատակային տեղադրում։

Այսպիսով, ստացված դիագրամը թույլ է տալիս սահմանել առավել ռիսկային գոտիները և հիմնավորել օդափոխության բացվածքների օպտիմալ տեղադրումը։

Ցույց տանք նաև մաթեմատիկորեն ընտրենք կենտրոնական հատվածի ներկայացուցիչ $P = (7,5)$ կետը։ Հաշվելով հեռավորությունը հետևյալ բանաձևով․

$$d = \sqrt{\left( x - x_{i})^{2} + (y - y_{i})^{2} \right.\ }$$

Հաշվենք հեռավորությունը (6, 6) և (9,5) կետերից․

$$d_{1} = \sqrt{\left( 7 - 6)^{2} + (5 - 6)^{2} \right.\ } = \sqrt{1 + 1} = \sqrt{2} = 1.41$$

$$d_{2} = \sqrt{\left( 7 - 9)^{2} + (5 - 5)^{2} \right.\ } = \sqrt{4} = 2
$$Ռիսկը (7,5) կետի համար կստանանք․

$${R(7,5) = \frac{1}{1.41} + \frac{1}{2}
}{R(7,5) = 0.709 + 0.5
}{R(7,5) = 1.209
}$$Ընտրենք անկյունային հատվածի ներկայացուցիչ $Q = (1,1)$ կետը և հաշվենք հեռավորությունը (2,2) և (4,10) կետերից.

$${d_{1} = \sqrt{\left( 1 - 2)^{2} + (1 - 2)^{2} \right.\ } = \sqrt{1 + 1} = \sqrt{2} = 1.41
}{d_{2} = \sqrt{\left( 1 - 4)^{2} + (1 - 1)^{2} \right.\ } = \sqrt{9} = 3}$$

Ռիսկը (1,1) կետի համար կստանանք․

$${R(1,1) = \frac{1}{1.41} + \frac{1}{3}
}{R(1,1) = 0.709 + 0.333
}{R(1,1) = 1.042}$$

Կատարված հաշվարկները և վերլուծությունը ցույց են տալիս, քանի որ $R(7,5) > R(1,1)$*  
*կենտրոնական հատվածում համակցված փոշային ազդեցությունը ավելի բարձր է, քան անկյունային հատվածում։

**Եզրակացություն**

Կատարված հաշվարկները և տարածական վերլուծությունը ցույց են տալիս, որ արտադրական տարածքում 8 փոշու աղբյուրների դեպքում երկրորդ կարգի Վորոնովի դիագրամի կիրառումը հանդիսանում է օդափոխության համակարգի տեղադրման տեխնիկապես հիմնավորված միջոցառում։

Ստացված արդյունքները ցույց են տալիս, որ․

- տարածքի կենտրոնական հատվածում ձևավորվում են բարձր ռիսկային գոտիներ,

- համակցված ազդեցության արժեքը առավելագույն է այն կետերում, որտեղ երկու աղբյուրների հեռավորությունները նվազագույն են,

- ռիսկի ֆունկցիայի արժեքները կենտրոնական գոտում գերազանցում են ծայրամասային հատվածների ցուցանիշները, աղբյուրների խտության աճը բերում է ազդեցության ինտենսիվության մեծացման։

Հետևաբար երկրորդ կարգի Վորոնովի դիագրամի կիրառումը փակ արտադրական տարածքներում հանդիսանում է նորմատիվային պահանջներին համապատասխան և ինժեներականորեն հիմնավորված լուծում, որն ուղղված է փոշու համակցված ազդեցության նվազեցմանը, օդափոխության համակարգի օպտիմալ տեղադրմանը և աշխատողների առողջության պաշտպանության բարձրացմանը։

# ԳԼՈՒԽ 5. ՎՈՐՈՆՈՅԻ ԴԻԱԳՐԱՄՆԵՐԻ և ՏԵՍՈՂԱԿԱՆ ՍԱՐՔԵՐԻ ԿԻՐԱՌՈՒՄԸ ԲՆԱՊԱՀՊԱՆԱԿԱՆ ԽՆԴԻՐՆԵՐԻ ԼՈՒԾՄԱՆ ՀԱՄԱՐ {#գլուխ-5.-վորոնոյի-դիագրամների-և-տեսողական-սարքերի-կիրառումը-բնապահպանական-խնդիրների-լուծման-համար}

Ժամանակակից բնապահպանական խնդիրների արդյունավետ լուծումը հնարավոր է միայն բազմամասնագիտական մոտեցման պայմաններում, որտեղ համադրվում են բնական գիտությունների, ինժեներական էկոլոգիայի և տեղեկատվական համակարգերի մեթոդները։ Բնապահպանական մենեջմենթի և էկոլոգիական անվտանգության ապահովման արդի փուլում շրջակա միջավայրում տեղի ունեցող գործընթացների խորքային ուսումնասիրությունը պահանջում է թվային և մաթեմատիկական գործիքակազմի անհապաղ ներդրում:

Տեսողական համակարգերը (վիդեոմոնիթորինգի ստացիոնար կայաններ, ջերմացույցներ, անօդաչու թռչող սարքեր) այսօր կարևորագույն դեր են խաղում բնապահպանական տվյալների հավաքագրման, ներկայացման և վերլուծության գործընթացում։ Դրանք թույլ են տալիս անընդհատ վերահսկել հսկայական տարածքներ, ֆիքսել էկոլոգիական խախտումները (ապօրինի ծառահատումներ, որսագողություն, հրդեհներ) և հավաքագրված բարդ տվյալները ներկայացնել հասկանալի և ինտուիտիվ ձևով՝ քարտեզների, գրաֆիկների և տարբեր տեսողական մոդելների միջոցով։

Մաթեմատիկական մոդելավորումը բնապահպանության ոլորտում հնարավորություն է տալիս նկարագրել իրական էկոհամակարգերը մաթեմատիկական հարաբերությունների միջոցով և ուսումնասիրել դրանց վարքը տարբեր, հաճախ՝ ճգնաժամային պայմաններում։ Այս մեթոդի օգնությամբ հնարավոր է գնահատել շրջակա միջավայրի վրա տարբեր գործոնների ազդեցությունը, ինչպես նաև կանխատեսել էկոլոգիական համակարգերի զարգացման հնարավոր սցենարները, ինչը կարևորագույն նախապայման է էկոլոգիական փորձաքննության և բնական ռեսուրսների կայուն կառավարման համար։

Բնապահպանական խնդիրների ուսումնասիրության ժամանակ մեծ նշանակություն ունեն գեոինֆորմացիոն համակարգերը (GIS) և գրաֆների տեսությունը, որոնք նախատեսված են տարածական տվյալների հավաքագրման և վերլուծության համար։ GIS համակարգերը հնարավորություն են տալիս տարբեր տեսողական սարքերից և սենսորներից ստացված տվյալները միավորել միասնական քարտեզագրական հարթակում՝ թույլ տալով տեսողականորեն գնահատել վտանգի տարածման արեալը:

- **Տեսողական սարքերի և արհեստական բանականության (ԱԲ) համակցված կիրառությունը էկոլոգիայում**

Լայնածավալ բնական համակարգերում զուտ տեսախցիկների տեղադրումը բավարար չէ, քանի որ հավաքագրվող տեսանյութերի հսկայական ծավալը մարդկային ռեսուրսով անհնար է վերլուծել իրական ժամանակում: Այդ իսկ պատճառով արդի ինժեներական էկոլոգիայում տեսողական համակարգերը սերտորեն ինտեգրվում են արհեստական ինտելեկտի և նեյրոնային ցանցերի տեխնոլոգիաների հետ։

Նեյրոնային ցանցերը թույլ են տալիս ավտոմատացնել տեսողական տվյալների մշակումը: Օրինակ, անտառային տարածքներում տեղադրված խելացի տեսախցիկները ունակ են վայրկյանների ընթացքում ճանաչել ծխի սյուները, անտառային հրդեհի օջախները կամ ապօրինի ծառահատում իրականացնող անձանց՝ անմիջապես ահազանգ ուղարկելով կառավարման կենտրոն:

Բացի այդ, տեսողական համակարգերը և նեյրոցանցերը անփոխարինելի են տրանսպորտային միջոցների էկոլոգիական վերահսկողության գործում։ Հայտնի է, որ ավտոտրանսպորտը հանդիսանում է շրջակա միջավայրի (օդային ավազանի և հողային ծածկույթի) աղտոտման հիմնական աղբյուրներից մեկը։ Խելացի տեսախցիկները կարող են ֆիքսել արգելոցներ կամ հատուկ պահպանվող կանաչ գոտիներ ապօրինի մուտք գործող ծանրաքաշ մեքենաները, վերլուծել տրանսպորտային հոսքերը և մոդելավորել դրանց արտանետումների ազդեցությունը բնական միջավայրերի էկոլոգիական վիճակի վրա:

Ավելին, էկոլոգիական սուր ճգնաժամերի կամ դժվարամատչելի վայրերի ուսումնասիրության համար, որտեղ ստացիոնար տեսախցիկներ հնարավոր չէ տեղադրել, ակտիվորեն կիրառվում են ռոբոտատեխնիկական միջոցներ և անօդաչու թռչող սարքեր (ԱԹՍ)։ Համալրված լինելով բարձր ճշգրտության օպտիկական սարքերով՝ դրանք կարող են իրականացնել տարածքի արագ պարեկություն, գնահատել հրդեհի մակերեսը կամ հայտնաբերել վտանգավոր թափոնների ապօրինի կուտակումները վերևից:

- **Վորոնոյի դիագրամների տեսական նշանակությունը տեսադիտարկման ցանցերի օպտիմալացման գործում**

Տարածական տվյալների վերլուծության ընթացքում հաճախ անհրաժեշտ է լինում մոդելավորել տվյալ տարածքում գտնվող օբյեկտների ազդեցության գոտիները, գնահատել դիտարկման կայանների տեղաբաշխման արդյունավետությունը կամ հայտնաբերել համակարգի «կույր գոտիները»: Այսպիսի խնդիրների լուծման համար իդեալական հաշվարկային գործիք են Վորոնոյի դիագրամները, որոնք մանրամասն ուսումնասիրվել են սույն դիպլոմային աշխատանքի մաթեմատիկական բաժնում: Վորոնոյի դիագրամը տվյալ տարածության դասավորման հատուկ տեսակ է, օրինակ՝  [մետրային տարածություն](https://hy.wikipedia.org/w/index.php?title=%D5%84%D5%A5%D5%BF%D6%80%D5%A1%D5%B5%D5%AB%D5%B6_%D5%BF%D5%A1%D6%80%D5%A1%D5%AE%D5%B8%D6%82%D5%A9%D5%B5%D5%B8%D6%82%D5%B6&action=edit&redlink=1), կախված է տրված օբյեկտների (ենթաբազմությունների) ընտանիքից ունեցած հեռավորությունից տարածությյան մեջ։ 

Բնապահպանական մոնիթորինգում Վորոնոյի դիագրամների կիրառության տեսական մոդելավորումը իրականացվում է հետևյալ ուղղություններով.

Ա) Ցանցի տրոհում և պարտականությունների բաշխում

Բ) Էկոլոգիապես խոցելի (կույր) գոտիների տեսական որոշումը

Գ) Բարձր կարգի դիագրամներ և վտանգի խաչաձև վավերացում

Տարածական մոդելավորման ժամանակ հաճախ պահանջվում է հաշվի առնել ոչ միայն մեկ ամենամոտ օբյեկտը, այլ նաև մի քանի մոտակա դիտակետերի համատեղ աշխատանքը։ Եթե մեկ տեսախցիկը ֆիքսում է անտառային հրդեհի ծուխ, այն ստանում է միայն ուղղության վեկտոր, բայց ոչ օջախի ճշգրիտ հեռավորությունը և կոորդինատը: Հրդեհի օջախի տրիանգուլյացիայի համար համակարգին անհրաժեշտ են տվյալներ առնվազն ևս երկու տեսախցիկներից: Այստեղ կիրառվում են աշխատանքում ուսումնասիրված k-րդ կարգի Վորոնոյի դիագրամները: Նախապես հաշվարկված բարձր կարգի դիագրամները համակարգին թույլ են տալիս ակնթարթորեն իմանալ վտանգի գոտուն հաջորդ 2 ամենամոտ տեսախցիկների ինդեքսները և ավտոմատ կերպով թեքել դրանք դեպի օջախը՝ խուսափելով կեղծ ահազանգերից:

- **Տեսողական սարքերի և մոնիթորինգային ցանցերի էլեկտրամագնիսական անվտանգությունը**

Էլեկտրամագնիսական դաշտերի երկարատև և ինտենսիվ ազդեցությունը կարող է էկոլոգիական լուրջ խնդիրներ հարուցել կենսահամակարգերի համար: Գիտական և կենսաբանական հետազոտությունները ցույց են տալիս, որ գերբարձրհաճախային ճառագայթման և հզոր բջջային աշտարակների անմիջական մերձակայքում կարող են գրանցվել կենդանական աշխարհի վարքագծային շեղումներ։ Մասնավորապես, ուժեղ ԷՄ դաշտերը կարող են խաթարել միջատների (հատկապես մեղուների) տարածական կողմնորոշումը, ապակողմնորոշել չվող թռչունների միգրացիոն համակարգերը և բացասական նյարդաբանական էֆեկտներ ունենալ մանր անտառային կաթնասունների վրա։

Այս ռիսկերը մինիմալացնելու նպատակով նախագծվող տեսադիտարկման և սենսորային ցանցերում պարտադիր է պահպանել էլեկտրամագնիսական անվտանգության հետևյալ կանոնները.

**Ցածր էներգիայի տեխնոլոգիաների գերակայություն.** Տեսողական մոնիթորինգին զուգահեռ, տարածքում դիրքի տեղորոշման և սենսորային ազդանշանների համար նախապատվությունը տրվում է BLE (Bluetooth Low Energy) տեխնոլոգիային: Դրա ճառագայթման հզորությունը անհամեմատ փոքր է ստանդարտ հաղորդիչներից և համարվում է էկոլոգիապես անվտանգ:

**Ազդանշանի իմպուլսային ռեժիմ.** Որպեսզի անտառային գոտին չենթարկվի մշտական ճառագայթային «աղմուկի», տեսախցիկները չպետք է անընդհատ վիդեոհոսք հեռարձակեն: ԱԲ ալգորիթմների շնորհիվ դրանք տվյալները կենտրոն պետք է փոխանցեն միայն շարժում, ապօրինի տրանսպորտ կամ անոմալիա (օրինակ՝ ծուխ) ֆիքսելու դեպքում (motion-triggered սկզբունքով): Սա կտրուկ կնվազեցնի ճառագայթման ակտիվ ժամանակահատվածը:

**Անվտանգ տեղակայում և գոտիավորում.** Տեսախցիկների հիմնական կայմերը, սերվերային հանգույցները և հզոր կապի անտենաները պետք է տեղադրվեն կենդանիների հիմնական բնադրման գոտիներից հեռու՝ հիմնվելով նախնական ՇՄԱԳ (շրջակա միջավայրի վրա ազդեցության գնահատում) փորձաքննության վրա։

Ամփոփելով նշենք, որ տեղեկատվական տեխնոլոգիաների, մաթեմատիկական մոդելավորման և տարածական վերլուծության մեթոդների համատեղ կիրառումը հնարավորություն է տալիս ռացիոնալ և ծախսարդյունավետ ուսումնասիրել բնապահպանական համակարգերը: Վորոնոյի դիագրամները և դրանց բարձր կարգի ածանցյալները հանդիսանում են օբյեկտիվ մաթեմատիկական գործիք գլոբալ բնապահպանական տեսադիտարկման ցանցերի օպտիմալացման, տեսադաշտի բաշխման և ճգնաժամային արձագանքման համար: Տեխնիկական և ռոբոտատեխնիկական միջոցների խելամիտ կիրառումը՝ զուգորդված արհեստական բանականության , էլեկտրամագնիսական անվտանգության նորմերի պահպանման և թափոնների ճիշտ կառավարման հետ, ապահովում է էկոլոգիական մոնիթորինգի խնդիրների կայուն և ժամանակակից լուծում:
